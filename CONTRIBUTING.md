# Contribuyendo a Fantasma Digital

¡Gracias por tu interés en contribuir a Fantasma Digital! Queremos que este proyecto sea lo más increíble posible y tu ayuda es bienvenida.

## Cómo contribuir

1.  **Reportar errores**: Si encuentras un bug, abre un "Issue" describiendo el problema y cómo reproducirlo.
2.  **Sugerir mejoras**: Si tienes una idea para una nueva funcionalidad o mejora estética, abre un "Issue" para discutirla.
3.  **Enviar Pull Requests**:
    - Haz un fork del repositorio.
    - Crea una rama para tu funcionalidad (`git checkout -b feature/nueva-funcionalidad`).
    - Realiza tus cambios y haz commit (`git commit -m 'Añade nueva funcionalidad'`).
    - Haz push a tu rama (`git push origin feature/nueva-funcionalidad`).
    - Abre un Pull Request.

## Estándares de Código

- Usamos **Astro** y **Tailwind 4**.
- Mantenemos la arquitectura de componentes organizada en `sections`, `ui` y `layout`.
- Priorizamos la estética y el rendimiento.
- Escribe mensajes de commit claros y en español (preferiblemente).

## Estética "Fantasma Digital"

Este proyecto tiene una identidad visual fuerte basada en el contraste, el minimalismo y el estilo de ingeniería de alto nivel. Por favor, intenta mantener coherencia con el diseño actual al sugerir cambios visuales.

¡Esperamos tus contribuciones!
